About Model Time

Model time allows you to simulate a full day in compressed time. This is useful for operating sessions where you want trains to run on a realistic schedule.

  • Start Time: The time when your model day begins (e.g., 6:00 AM)
  • Time Multiplier: How fast time moves relative to real time
  • Example: With 12x multiplier, 1 real hour = 12 model hours
  • Reset: Resets the model time back to the start time
